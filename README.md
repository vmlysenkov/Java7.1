* счетчик *INSTRUCTION* предоставляет информацию о количестве кода, который был выполнен или пропущен
* счетчик *BRANCH* вычисляет охват веток для операторов if и switch. Эта метрика подсчитывает общее количество таких ветвей в методе и определяет количество выполненных или пропущенных ветвей
* счетчик *LINE* рассчитывает информацию о покрытии для отдельных строк. Исходная строка счтитается выполненной, когда была выполнена хотя бы одна команда, назначенная этой строке.
